Javascript自定义事件

January 18, 2015

自定义事件在很多地方都会用到，然后跟着书实例了几次，然后详细注释，也算给自己做了一个笔记。

 1 //定义一个自定义事件类
 2 //定义一个实例属性用以单独为每一个实例创建独立的事件集
 3 function EventTarget() { 4 
 5     this.handles = {}; 6 
 7 };

30 }, 31

37          if(!event.target) { 38              event.target = this; 39 }; 40

82                  if(handles\[i\] === handle) { 83
